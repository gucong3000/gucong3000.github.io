jQuery.CSS3插件

css("position")

概述

访问第一个匹配元素的样式position。

IE6 或 IE 怪癖模式下，为top设置了expression且position为absolute的元素会返回"fixed"。

示例

alert($("#qqmsg").css("position"));

css("position", value)

概述

在所有匹配的元素中，设置样式position的值。

IE6 或 IE 怪癖模式下通过jQuery.css3插件实现fixed，并修正position设置为absolute或fixed时，设置width或height为"xx%"时宽度和高度有时不正确的问题。

可用值

static : 无特殊定位，对象遵循HTML定位规则。

absolute : 将对象从文档流中拖出，使用left，right，top，bottom等属性进行绝对定位。而其层叠通过z-index属性定义。此时对象不具有边距，但仍有补白和边框。

relative : 对象不可层叠，但将依据left，right，top，bottom等属性在正常文档流中偏移位置。

fixed:与absolute类似，但不随滚动条滚动。

示例

$("#qqmsg").css("right","18px"); $("#qqmsg").css("bottom",";5px"); $("#qqmsg").css("position","fixed");
