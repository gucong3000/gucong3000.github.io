jQuery.CSS3插件

css("background", value)

概述

在所有匹配的元素中，设置样式background的值。

可用值

该属性是复合属性，使用空格分隔各值即可。请参阅各参数对应的属性。

示例

$(document.body).css("background","url('chess.png') gray 50% repeat fixed");

css("background-image")

概述

访问第一个匹配元素的样式background-image。

示例

alert($(document.body).css("background-image"));

css("background-image", value)

概述

在所有匹配的元素中，设置样式background-image的值。

可用值

none:默认值。无背景图

url( url ):使用绝对或相对 url 地址指定背景图像

linear-gradient(<start-point>, <color-start>, <color-stop>):线性渐变颜色。start-point指渐变开始的方向，可取top、left、right、bottom,<color-start>, <color-stop>为开始和结束的颜色。

jQuery.CSS3插件会统一不同浏览器下的线性渐变语法，使用统一的语法即可。

示例

$(document.body).css("background","linear-gradient(top, rgba(255, 255, 0, 0.5), rgba(127, 127, 0, 0.5))");

css("background-color")

概述

访问第一个匹配元素的样式background-color。

示例

alert($(document.body).css("background-color"));

css("background-color", value)

概述

在所有匹配的元素中，设置样式background-color的值。

可用值

<name>：使用颜色名。

#RGB：三位十六进制正整数。取值范围为：0 - F。

#RRGGBB：三个两位十六进制正整数。取值范围为：00 - FF。

rgb(r, g, b)：r,g,b表示红，绿，蓝的正整数或百分数数值。

rgba(r, g, b, a)：r,g,b表示红，绿，蓝的正整数或百分数数值，a表示不同明度的百分数数值或0-1的小数。

IE通过就Query.CSS3插件实现rgba(r, g, b, a)。

示例

$(document.body).css("background-color","red"); $(document.body).css("background-color","#0FF"); $(document.body).css("background-color","#0000FF"); $(document.body).css("background-color","rgb(0, 255, 255)"); $(document.body).css("background-color","rgba(0, 255, 255, 0.6)");
